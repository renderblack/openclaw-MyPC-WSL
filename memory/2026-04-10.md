### 关键经验总结

#### 1. 代理配置
```bash
# WSL2 需要通过 Windows 代理访问外网
--proxy http://172.18.176.1:7890
```
- WSL2 的 DNS 是 `172.18.176.1`
- 代理地址是 `172.18.176.1:7890`

#### 2. 无头 vs 露头模式
| 模式 | 速度 | 反爬检测 | 适用场景 |
|------|------|---------|---------|
| 无头 (headless) | 快 | 会触发 | 简单页面、不需要登录 |
| 露头 (headful) | 慢 | 不触发 | 扫码登录、反爬严格网站 |

#### 3. 常用命令
```bash
# 关闭所有会话
agent-browser close --all

# 带代理打开网站
agent-browser open --proxy http://172.18.176.1:7890 https://www.bilibili.com

# 截图
agent-browser screenshot

# 查看页面元素
agent-browser snapshot -i

# 点击元素
agent-browser click <ref>

# 查找并点击
agent-browser find text "登录" click
```

#### 4. 截图发送微信
```bash
# 截图保存路径
~/.agent-browser/tmp/screenshots/

# 复制到workspace
cp <截图路径> ~/.openclaw/workspace/screenshots/

# 通过 message 工具发送到微信
message(action=send, channel=openclaw-weixin, filePath="路径", target="o9cq80xO-9exHj-UMC1xQUSNgK5g@im.wechat")
```
