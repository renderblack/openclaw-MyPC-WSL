**发现**：已有 `hybrid-search.ps1` 脚本，路径：`~/.openclaw/workspace/scripts/hybrid-search.ps1`

**脚本功能**：
- `Everything/es.exe` + `ripgrep/rg.exe` 组合
- 支持三种模式：`name`（文件名）、`content`（内容）、`all`（混合）
- 支持 `-ext` 指定扩展名，`-MaxResults` 限制结果数

**环境已确认**：
| 组件 | 路径 |
|------|------|
| Everything CLI | `C:\ProgramData\chocolatey\lib\es\tools\es.exe` |
| ripgrep | `C:\ProgramData\chocolatey\bin\rg.exe` |

**使用示例**：
```powershell
# 搜索文件名 + 内容
.\hybrid-search.ps1 "关键词" -dir D:\notes

# 只搜文件名
.\hybrid-search.ps1 "报告" -dir C:\ -type name

# 只搜 .md 文件内容
.\hybrid-search.ps1 "OpenClaw" -dir C:\Users\Administrator -ext md -type content
```
