## 🎯 需求确认

### 约束条件
- **无 API Key**：纯本地/免费方案
- **环境**：Windows 10 Native + PowerShell
- **输出方式**：通过 QQ 发送语音给老大，老大手机接收播放

### 技术选型

| 功能 | 方案 | 说明 |
|------|------|------|
| **TTS 语音合成** | sherpa-onnx-tts | 纯离线，无需网络 |
| **STT 语音识别** | openai-whisper | 本地 Whisper 模型 |

### Whisper 模型选择
- **优先下载**: base (~140MB) - 平衡选择
- **备用下载**: small (~500MB) - 准确率高
- 两个模型都保留，按需切换

### 语音用途
1. **播报/讲故事** (TTS) - 主要场景
2. **语音指令** (STT) - 通过语音控制AI
3. **语音通话** (voice-call) - 后续再看

## 🔧 技术备注

### sherpa-onnx-tts 安装要点
- 需要下载 `sherpa-onnx` Python 包
- 需要下载 VITS 中文模型（vits-zh）
- 输出格式：wav / mp3

### openai-whisper 安装要点
- 需要安装 `openai-whisper` Python 包
- 模型文件：~140MB (base) / ~500MB (small)
- 支持格式：mp3, wav, m4a, mp4

### QQ 语音发送
- 使用 qqbot-media skill
- 语音格式：silk（QQ 专用）
- 需要 ffmpeg 转换格式
