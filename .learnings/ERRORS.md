# Errors

Command failures and integration errors.

**Format**:
```markdown
- **[Date]** `[Error Type]` **Description**
  - **Context**: What was happening?
  - **Root Cause**: Why did it fail?
  - **Fix**: How was it resolved?
  - **Prevention**: How to avoid in future?
```
